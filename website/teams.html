How it works

1

Install on each machine

Every developer installs the CLI. Works with Claude, Cursor, or any AI assistant.

2

Connect team events

$ ginko login && ginko team connect

Authenticate and link your team. Events sync automatically. Pro

3

See shared insights

$ ginko start

Every session loads with team context. Patterns, gotchas, sprint tasks — all there.
