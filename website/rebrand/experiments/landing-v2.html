FAQ

Questions?

Is my code safe? +
Yes. All context is stored locally in your .ginko folder. Nothing leaves your machine unless you push to a remote repository.
Does it work with Cursor, Copilot, or Claude? +
Yes. ginko CLI is model-agnostic and works with any AI tool that can read your project files.
How is this different from CLAUDE.md? +
CLAUDE.md is static. ginko captures dynamic session context: decisions, patterns, and handoff notes. It's living memory.
